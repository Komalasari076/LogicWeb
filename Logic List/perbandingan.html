Perbandingan Array

Masukkan dua array untuk dibandingkan. Aplikasi ini akan memeriksa apakah kedua array memiliki panjang yang sama dan elemen yang sama. Hasil perbandingan akan ditampilkan di konsol browser.

Cara Penggunaan:

  1. Masukkan elemen array pertama di kolom input "Array Pertama".
  2. Masukkan elemen array kedua di kolom input "Array Kedua".
  3. Klik tombol "Bandingkan Array".
  4. Akan muncul alert hasil perbandingan
