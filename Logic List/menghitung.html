Menghitung Karakter

Masukkan teks ke dalam kolom input, dan sistem akan secara otomatis menghitung jumlah kemunculan setiap karakter di dalam teks tersebut.

Cara Penggunaan:

  1. Masukkan teks ke dalam kolom input
  2. Klik tombol Hitung Karakter
  3. Akan muncul alert hasil perhitungan karakter

Cara Penggunaan:

  1. Masukkan teks ke dalam kolom input Text Pertama
  2. Masukkan teks ke dalam kolom input Text Kedua
  3. Klik tombol Hitung Karakter
  4. Akan muncul alert hasil perhitungan karakter
