Mengurutkan Angka

Pilih mode input tunggal atau ganda, masukkan angka-angka, dan program akan mengurutkan angka secara otomatis.

Cara Penggunaan:

  1. Masukkan angka ke dalam kolom input.
  2. Klik tombol "Urutkan Angka".
  3. Hasil urutan angka akan muncul pada alert.

Cara Penggunaan:

  1. Masukkan angka ke dalam kolom input "Angka Pertama".
  2. Masukkan angka ke dalam kolom input "Angka Kedua".
  3. Klik tombol "Urutkan Angka".
  4. Hasil gabungan urutan angka akan muncul pada alert.
